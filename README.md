# unique-search-cli

A command-line utility for developers and power users to search text patterns in files and directories with optional uniqueness filtering, regex support, and case-insensitivity. Provides clear, reliable, and efficient output.

## Install

Clone or download this repository, then:

```sh
npm install
```

## Usage

```
unique-search <query> [options] [path]
```

_Basic skeleton project structure. Full usage will be documented in subsequent versions._
